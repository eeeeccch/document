특정 영역을 마스크하거나 클리핑하여 요소를(부분적으로 또는 전체적으로) 숨김


주요 속성
- `mask-clip` 마스크가 적용될 영역 설정
- `mask-composite` 여러 마스크 조합 방법 설정
- `mask-image` 마스크에 사용할 이미지 지정
- `mask-mode` 마스크 모드 설정
- `mask-origin` 마스크의 원점 설정
- `mask-position` 마스크 위치 조정
- `mask-repeat` 마스크 이미지의 반복 여부 설정
- `mask-size` 마스크 크기 조정
